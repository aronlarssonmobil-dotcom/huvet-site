# Trafik

Första körbara MVP-shellen för körkortsplattformen.

## Vad som finns nu

- Polerad landningssida för Trafik
- Tydlig positionering kring smart teori, svaghetsanalys och dagsplan
- Interaktiv demo med frågor, direkt feedback och live-analys
- Svensk metadata och enkel navigation

## Kör lokalt

```bash
npm install
npm run dev
```

Öppna sedan `http://localhost:3000`.

Om port 3000 är upptagen väljer Next automatiskt en annan port och visar den i terminalen.

## Bygg

```bash
npm run build
```
